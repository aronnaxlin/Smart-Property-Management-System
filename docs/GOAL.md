# 设计内容：

- 设计开发一个简单的“小区物业管理系统”，并实现基本的管理、查询功能。
- 能够实现小区用户基本信息的录入、修改、删除等操作；
- 能够实现小区物业费、取暖费等费用的管理，对费用的收取进行记录，对逾期未缴费的用户能够进行查询；
- 能实现小区电卡、水卡费用购买的管理工作，对未交物业费、取暖费的用户，此项功能失效，待缴纳物业费、取暖费的用户才能继续购买；
- 能方便的对小区用户情况进行查询，可以根据房号、姓名等多种方式查询。

# 设计要求：

- 使用Java及相关语言完成开发，数据库设计要求有ER图、数据字典、测试数据等，交互界面不能是控制台方式；
- 按照软件工程的思想完成系统开发过程；
- 最终的软件系统要求数据充实、界面友好、使用方便。
